zabbix-server无法启动

zabbix5.0配置了主机自动注册后，主机注册到一半的时候(总数150+)，zabbix server挂了，然后日志信息如下，各种查，在参考链接中有提到可能是由于 CacheSize 的问题，调大 zabbix server中 CacheSize的值，然后重启zabbix server就可以了
