# ansible模块-服务模块

**ansible查看帮助方法**

```python
# 查看所有模块帮助信息
ansible-doc -l   

# 指定查看某个模块参数用法
ansible-doc copy  
```



## 1 service服务模块

```shell
ansible all -m service -a "name=httpd state=stopped enabled=yes"
```
