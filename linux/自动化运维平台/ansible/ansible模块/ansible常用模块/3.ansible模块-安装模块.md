# ansible模块-安装模块

**ansible查看帮助方法**

```python
# 查看所有模块帮助信息
ansible-doc -l   

# 指定查看某个模块参数用法
ansible-doc copy  
```



## 1 yum安装软件模块

```python
ansible all -m yum -a "name=httpd state=installed"
```
